JAVASCRIPT:

fetchStatus() ogni 2s: aggiorna pallino e current task
fetchPending() ogni 3s: popola lista coda
fetchDone() ogni 5s: popola completati
fetchLogs() ogni 2s: aggiorna log box con scroll

control(action): POST /api/control/action

deleteTask(filename): DELETE + fetchPending

addTask(event): preventDefault, POST /api/tasks/add, reset form

toggleFilepath(): mostra/nascondi filepath input

importTasks(): 
  leggi textarea importJson
  parse JSON
  POST /api/tasks/import con body
  mostra risultato (N task importati)
  fetchPending()

handleFileDrop(event):
  preventDefault
  leggi file trascinato
  metti contenuto in textarea importJson

setupDragDrop():
  area = getElementById importArea
  area.ondragover: preventDefault, addClass dragover
  area.ondragleave: removeClass dragover
  area.ondrop: handleFileDrop

document.addEventListener DOMContentLoaded:
  setupDragDrop()
  chiama tutti i fetch
  avvia tutti i setInterval
