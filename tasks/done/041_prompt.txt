JAVASCRIPT in <script>:
- async function fetchStatus() chiama GET /api/status, aggiorna statusDot classe e statusText, aggiorna currentTask section
- async function fetchPending() chiama GET /api/tasks/pending, popola pendingList con task-item, aggiorna pendingCount
- async function fetchDone() chiama GET /api/tasks/done, popola doneList
- async function fetchLogs() chiama GET /api/logs, aggiorna logBox innerHTML, scrolla in basso
- async function control(action) chiama POST /api/control/action, poi fetchStatus
- async function deleteTask(filename) se confirm, chiama DELETE, poi fetchPending
- async function addTask(event) preventDefault, leggi form, chiama POST /api/tasks/add con JSON, resetta form, fetchPending
- function toggleFilepath() mostra/nascondi filepathRow se type==file
- setInterval fetchStatus 2000
- setInterval fetchPending 3000
- setInterval fetchDone 5000
- setInterval fetchLogs 2000
- chiama tutte le fetch all'avvio
